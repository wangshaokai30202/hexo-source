# 闲言碎语功能介绍 — 类似朋友圈的动态发布系统

“闲言碎语”是我博客中的一个特色功能，它有点类似于朋友圈，可以随时随地发布各种内容，包括文本说说、图片、视频、音乐、外部链接等，非常适合记录生活点滴和分享兴趣爱好。
![](https://wangshaokai040813blog.cfd/wangshaokai/20251125225303827.webp)
![](https://wangshaokai040813blog.cfd/wangshaokai/20251125225303827.webp)

---

## 一、功能简介

闲言碎语的主要功能包括：

- **发布文本说说**：记录生活、心情、日常琐事等。
- **嵌入图片**：支持单张或多张图片的展示。
- **嵌入视频**：支持 B 站视频或本地视频播放。
- **嵌入音乐**：支持自定义歌单，可使用 APlayer 播放。
- **嵌入链接**：可分享文章、番剧或其他外部网页。
- **首页展示**：可设置是否显示在首页，以及显示条数限制。

通过这些功能，用户可以像在朋友圈一样自由地记录和展示内容。

## 二、使用方式

闲言碎语的内容存储在博客目录下的 `_data/essay.yml` 文件中，格式为 YAML。下面将各类型内容整理成示例，方便你快速添加新的动态。

1. 文本说说
```
- content: 功能测试，嵌入作者
  date: 2025/11/22
  address: 齐齐哈尔
  from: 阿凯
```

2. 外部链接说说
```
- content: 功能测试，嵌入外部链接
  date: 2025/11/22
  link: https://www.bilibili.com/
  address: 齐齐哈尔
```

3. 图片说说
```
- content: 功能测试，嵌入图片说说
  date: 2025/11/22
  image:
    - https://bu.dusays.com/2023/04/09/643293997b92b.jpeg
    - https://bu.dusays.com/2023/04/09/64329399e285d.webp
    - https://bu.dusays.com/2023/04/09/64329399aa3bc.webp
    - https://bu.dusays.com/2023/04/09/6432939996dd7.webp
  address: 齐齐哈尔
```

4. 视频说说
```
- content: 功能测试，B 站视频嵌入测试
  date: 2025/11/22
  video:
    - https://player.bilibili.com/player.html?isOutside=true&aid=115472611941838&bvid=BV1h915B6EDy&cid=33582811973&p=1
```

5. 音乐说说
```
- content: 音乐支持自定义歌单
  date: 2023/01/02
  link: https://hexo.anheyu.com/music/?id=7269231710&server=tencent
```

6. 综合示例
```
- content: 支持了 Accesskey 快捷键，可以直接按下 shift + ? 查看快捷键选项。
  date: 2023/07/01
  video:
    - https://cdn.jsdelivr.net/npm/anzhiyu-blog-static@1.0.0/video/%E9%A3%8E%E8%BD%A6%E6%A0%B7%E5%BC%8F%E6%95%88%E6%9E%9C%E9%A2%84%E8%A7%88.mp4
  image:
    - https://img02.anheyu.com/adminuploads/1/2023/07/01/64a033cb2c21e.webp!blogimg
  address: 齐齐哈尔市
  from: 阿凯
  link: /posts/e140.html
```

💡 提示：
  content：说说内容
  date：发布时间
  image：图片数组，可添加多张
  video：视频数组，可添加 B 站或本地视频
  link：外部链接或文章链接
  aplayer：音乐播放设置
  from：作者
  address：地点

## 三、成品展示
在博客中，“闲言碎语”模块会在首页或单独页面以卡片形式展示每条说说，支持多媒体内容播放和外部链接跳转。例如：

文本说说：显示作者、时间、地点与文字内容。

图片说说：多张图片展示，支持轮播或瀑布流。

视频说说：直接嵌入播放器播放视频。

音乐说说：在页面播放音乐，可自定义歌单。

链接说说：点击跳转到指定网页或文章。
